# captive-admin-ui

## 安装依赖
```
npm install
```

### 开发调试
```
npm run serve
```

### 打包
```
npm run build
```
- 进行将各个模块打包为各自的`.js`文件，样式文件也同时抽离处理为同名的`.css`
```
npm run build_all
```
- 将所有的功能打包为`index.js`和`index.css`
